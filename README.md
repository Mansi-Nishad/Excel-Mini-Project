# 🌍📊 Excel Mini Project — Country Flag Selector

## 🚀 Project Overview
This project is an interactive Excel dashboard that allows users to select a country and dynamically view its corresponding flag in real time.

It demonstrates my skills in building clean, structured, and visually interactive dashboards using Microsoft Excel.

---

## ✨ Key Features
- ⚡ Dynamic Dashboard with real-time updates  
- 🌍 Country Flag Selector (interactive output)  
- 📊 Interactive Charts & Visualizations  
- 🎛️ Slicers for better user experience  
- 🧠 Clean and structured data handling  

---

## 🛠 Tools & Technologies Used
- Microsoft Excel  
- Pivot Tables  
- Slicers  
- Data Validation  
- Dynamic Charts  
- Excel Formulas (IF, VLOOKUP / XLOOKUP)
